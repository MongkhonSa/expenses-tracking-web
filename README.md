## Required

- node 12.0.0+

- npm 7.0.0+

## 1. Install the package

```bash
npm install
# or
yarn  install
```

## 2. Test

```bash
npm run test
```

```bash
npm run test:coverage
```

## 3. Build Package (for production server)

```bash
npm run build
# or
yarn build
```

## 4. Getting Started

run the development server:

```bash
npm run dev
# or
yarn dev
```

run the production server:

```bash
npm run start
# or
yarn start
```
